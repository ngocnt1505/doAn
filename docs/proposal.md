| Topic:  3D House Defense Game Implemented with WebGL and Three.js |
| :---- |
| **Description:**  The goal of this project is to design and implement a simple interactive 3D mini-game that runs directly in a web browser using Three.js and WebGL. In this game, the player must protect a small house from incoming cute monsters. When entering the website, the player can view the entire scene including the house, yard, fence, monsters, and weapon.  During gameplay, monsters randomly move toward the yard area outside the house. The player aims using the mouse and clicks to shoot bullets at the monsters. The bullet has a travel time, requiring the player to predict the monster's movement to successfully hit the target. The game demonstrates basic concepts of computer graphics such as 3D scene construction, lighting, texture mapping, animation, and object interaction. The system is implemented as a web-based application so that it can run on modern browsers without installing additional software.  |
| **Expected outcomes:**  The expected outcome of this project is a fully functional browser-based 3D mini-game. The final system will include a complete 3D environment containing a house, yard, fence, monsters, and weapon. The player will be able to interact with the game by selecting a weapon, aiming with the mouse, and shooting monsters. The gameplay will be organized into three waves of monsters: Wave 1: 3 monsters Wave 2: 5 monsters Wave 3: 7 monsters The game will include three types of monsters, each with different movement speeds and visual appearances. The player will start the game with one basic weapon. After completing each wave, the player will unlock a new weapon.  The game will include the following conditions: Win condition: the player wins after defeating all monsters in the third wave. Lose condition: the player loses if a monster reaches the house. The project will demonstrate the application of several fundamental computer graphics techniques including 3D object rendering, lighting models, texture mapping, and basic animation.  |
| **Used Methods and Techniques:** The project will apply several computer graphics and web development techniques. The system will use Three.js as the main graphics library built on top of WebGL to render 3D scenes in the browser. The following techniques will be implemented: 3D scene creation and rendering Perspective camera setup Lighting models (Ambient Light and Directional Light), shadow casting Polygon mesh representation of 3D objects Texture mapping for environment objects Collision detection between bullets and monsters Mouse interaction and raycasting for aiming and shooting Random Walk and Target Seeking for monster movement, Projectile Motion for Bullet Threejs geometry for simple 3D asset, import GLTF files for complex 3D asset These techniques demonstrate how fundamental computer graphics concepts can be applied in a simple interactive web-based game. |
